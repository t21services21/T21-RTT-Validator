# Pathway 1 – Unit Checklists

Use this document to track learner progress. Each unit has **knowledge** and
**skills** checkpoints.

---

## Unit 1 – Introduction to Data & the Role of the Data Scientist

**Knowledge**
- [ ] Can explain in simple language what data science is.
- [ ] Can describe key data roles (analyst, scientist, engineer, ML engineer).
- [ ] Understands the main stages of CRISP-DM.
- [ ] Is aware of ethics, privacy, bias and responsible AI.

**Skills**
- [ ] Can open a notebook and run cells.
- [ ] Can load a CSV file and inspect rows/columns.
- [ ] Can describe what one row represents in a dataset.

---

## Unit 2 – Python Programming for Data

**Knowledge**
- [ ] Recognises Python basic types (int, float, str, bool, list, dict).
- [ ] Understands the purpose of `if` statements, loops and functions.

**Skills**
- [ ] Can write and run simple Python scripts.
- [ ] Can use lists and dictionaries to store small collections of data.
- [ ] Can write a basic function with parameters and a return value.

---

## Unit 3 – Working with Data using Pandas & NumPy

**Knowledge**
- [ ] Understands what a DataFrame is and why it is useful.
- [ ] Knows the difference between row-wise and column-wise operations.

**Skills**
- [ ] Can load CSV data into Pandas.
- [ ] Can handle missing values and duplicates using sensible strategies.
- [ ] Can create new columns (e.g. year, month) from existing data.
- [ ] Can produce grouped summaries (e.g. by region, category).

---

## Unit 4 – SQL & Relational Databases for Analysis

**Knowledge**
- [ ] Understands tables, rows, columns and keys.
- [ ] Knows what `SELECT`, `FROM`, `WHERE`, `JOIN`, `GROUP BY` do.

**Skills**
- [ ] Can write basic SELECT queries with filters.
- [ ] Can join at least two tables correctly using keys.
- [ ] Can aggregate data by group with SUM/COUNT and explain the result.

---

## Unit 5 – Statistics & Probability for Data Science

**Knowledge**
- [ ] Understands mean, median, variance, standard deviation.
- [ ] Understands correlation vs causation at a basic level.
- [ ] Knows the idea of a hypothesis test and p-value (high level).

**Skills**
- [ ] Can compute descriptive statistics in Python/Pandas.
- [ ] Can interpret basic A/B test results (conversion rates, lift).

---

## Unit 6 – Data Visualisation & Storytelling

**Knowledge**
- [ ] Knows when to use bar, line, and scatter plots.
- [ ] Understands why clutter and 3D effects can hurt clarity.

**Skills**
- [ ] Can build clear charts with labels and titles.
- [ ] Can write a short narrative explaining what a chart shows and why it
      matters.

---

## Unit 7 – Foundations Capstone Project

**Knowledge**
- [ ] Understands the end-to-end flow of a small data project.

**Skills**
- [ ] Has completed a capstone project using at least one dataset.
- [ ] Can explain problem, data, method and results in plain language.
- [ ] Has produced artefacts suitable for a portfolio (notebook, charts,
      short report).

---

## Overall Readiness (Tutor Use)

- [ ] Learner demonstrates consistent attendance and engagement.
- [ ] Learner can explain their work without reading from notes.
- [ ] Learner is ready to progress to **Pathway 2 – Professional Data
      Science**.
